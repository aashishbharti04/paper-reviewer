What it does

📄

PDF & DOCX ingest

Pulls text and metadata straight from the submitted manuscript.

🗂️

Publisher rules engine

Assigns each paper to the right publisher bucket using editable rules.

🤖

AI-text detection

Flags machine-generated writing before it reaches reviewers.

🔍

Plagiarism check

Compares against a local corpus and optional deep web check.

📊

CMT-style Excel export

Appends results to a Microsoft CMT–schema sheet, columns preserved.

🖥️

Web + desktop

FastAPI dashboard, native Edge app window, or legacy Tkinter UI.

Get started

Install — download PaperReviewer-Setup.exe and run it. No Python needed.
Add a free LLM key — open the Settings tab and paste a Groq, OpenRouter, or local Ollama key.
Review — upload a paper, let it score and bucket, then export to Excel.

Run from source

pip install -r requirements.txt
copy config.json.example config.json   # add your provider key
python -m paper_reviewer_web.app        # open http://localhost:8765
